# Трекер Ули

Локальный HTML-дашборд: привычки, счётчики, разовые задачи, события, регулярка.

## Файлы
- `index.html` — весь дашборд (один файл, inline CSS+JS)
- `lib/chart.min.js` — Chart.js 4.4.0 (локально, офлайн)

## Где хранятся данные
**Основное место** — `~/Documents/Personal/Tracker/data.json`. Туда же пишет Telegram-бот (Personal/Health/bot/). Трекер автоматически синхронизируется с этим файлом через локальный HTTP-сервер бота на `localhost:8765`.

**Резервная копия** — localStorage браузера (ключ `tracker_v1`). Используется как фолбек если сервер бота не запущен.

При первом открытии подгружаются SEED-данные (привычки, задачи, события) — потом всё кликабельно и сохраняется в data.json.

## Как открыть
**Лучший способ** (синхронизация с ботом):
```
open -a "Google Chrome" http://localhost:8765/
```
Бот должен быть запущен (`Personal/Health/bot/start_bot.command`).

**Без бота** (только локально):
```
open -a "Google Chrome" ~/Documents/Personal/Tracker/index.html
```
