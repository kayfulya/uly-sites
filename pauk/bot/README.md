# 🕷 Бот Реестра Очков Уважения

Telegram-бот, который ловит команду `/respect` в чатах Совета и пишет начисления прямо в `../data.json` и `../data.js`. Дашборд `index.html` сразу видит новые записи — просто обнови страницу.

## Что умеет

- `/respect <кому> <сколько> <причина>` — начислить очки
- `/balance <кого>` — баланс одного
- `/top` — топ 10 по балансам
- `/ustav` — правила
- `/start` или `/help` — справка

## Кто может начислять

Только трое: **Уля, Таня, Толя**. Их Telegram user_id зашиты в `config.json`. Любые `/respect` от других — реплай «Не уполномочен» и игнор.

## Кап 10 000

Если в команде сумма больше 10 000 — бот сам отрежет до потолка и припишет в реестре, что заявлено было больше.

## Запуск — первый раз

### 1. Получить токен бота

1. Открой Telegram, найди **@BotFather**
2. `/newbot` → придумай имя (например `Pauk Respect Bot`) и username (например `PaukRespectBot`)
3. BotFather пришлёт токен вида `123456:ABC-DEF...` — это секрет, никому не показывай

### 2. Получить user_id троих лидов

1. Каждый (Уля, Таня, Толя) пишет **@userinfobot** команду `/start`
2. Бот отвечает числом — это и есть user_id
3. Соберёшь себе три числа

### 3. Заполнить config.json

```bash
cd ~/Documents/Personal/Pauk-respect/bot
cp config.example.json config.json
```

Открой `config.json` в редакторе и подставь токен и три user_id:

```json
{
  "bot_token": "123456:ABC-DEF...",
  "allowed_user_ids": {
    "ulya": 111111111,
    "tanya": 222222222,
    "tolya": 333333333
  },
  "cap": 10000
}
```

### 4. Установить зависимости (один раз)

```bash
cd ~/Documents/Personal/Pauk-respect/bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 5. Запустить

```bash
./run.sh
```

или вручную:

```bash
source venv/bin/activate
python bot.py
```

Увидишь `🕷 Бот запущен.` — значит работает.

### 6. Добавить бота в чаты

В Telegram открой каждый нужный чат (General, Болталка, Моторы, Лидное и т.д.) → Управление → Администраторы (или просто Добавить участника) → найди своего бота по username → добавь.

Для команд **бота не обязательно делать админом** — достаточно простого участника.

### 7. Проверить

В любом чате напиши: `/respect Вася 100 тест` — должен ответить «✅ +100 → 🔧 Вася...» и в `data.json` появится новая запись.

## Имена и алиасы

Бот понимает имена в свободной форме — «Вася», «Васе», «Васю» — всё ведёт к `vasya`. Все варианты лежат в `aliases.json`. Если нужно добавить — просто допиши строку и перезапусти бота.

Несколько получателей через запятую без пробела: `/respect вася,таня,лёша 500 за движ`.

## Если что-то сломалось

- **«❌ Не уполномочен»** — твой user_id не в `config.json`. Проверь, что подставил правильный.
- **«❌ Не знаю кто это»** — добавь алиас в `aliases.json`.
- **Бот не отвечает в группе** — проверь, что добавила его в чат, и команды пишешь через `/` (а не текстом).
- **Несколько ботов в чате с одинаковой командой** — пиши `/respect@PaukRespectBot ...`.

## Как остановить

В терминале где запущен бот: `Ctrl+C`.
